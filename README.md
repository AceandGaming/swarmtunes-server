# Swarmtunes - Server

The server-side code that runs Swarmtunes. It provides the backend API for the main website at [api.swarmtunes.com](https://api.swarmtunes.com).

See [`Swarmtunes - Client`](https://github.com/AceandGaming/Swarmtunes-client) for the main website and client code.

To setup and run a local server, see the guide at [Contributing](https://github.com/AceandGaming/swarmtunes-server?tab=contributing-ov-file).
